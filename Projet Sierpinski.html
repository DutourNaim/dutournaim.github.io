Présentation du Projet Sierpinski.
Mon projet NSI Programmation

L’utilisation des "courbes" représentées sous la forme de chaines de caractères demande de définir deux informations pour tracer un tapis de Sierspinski :

1- des mots initiaux,
2- des règles de remplacement.

Dans l’énoncé du projet, les mots initiaux et les règles de transformation sont :

Le mot de départ = "F-G-G"
La règle 1 = ("F","F-G+F+G-F") c’est-à-dire F→ F-G+F+G-F
La règle 2 = ("G","GG") c’est-à-dire G→GG

En appliquant les deux règles de transformation, les n-itération nous donnent :

pour n = 0,
la fonction renvoie le mot de départ F-G-G,
pour n = 1,
la fonction renvoie (F-G+F+G-F) – (GG) – (GG)
pour n = 2, la fonction renvoie :
(F-G+F+G-F) - (GG) + (F-G+F+G-F) + (GG) - (F-G+F+G-F)
etc

Ensuite, l’énoncé du projet nous donne les informations de déplacement :
F et G signifient tous les deux un déplacement de 10 pixels
+ signifie tourner à gauche d'un angle de 120°
- signifie tourner à droite d'un angle de 120°
